# CodeLeap Frontend Test

This project is a frontend implementation of the CodeLeap technical test.

The application allows users to create, read, update and delete posts using the CodeLeap API.

## Technologies

- React
- Vite
- JavaScript
- CSS

## Features

- User signup (username)
- Create posts
- List posts from API
- Edit posts
- Delete posts
- Conditional actions (edit/delete only for the post owner)

## API

The project consumes the public API:

https://dev.codeleap.co.uk/careers/

## How to run the project

Clone the repository:
https://github.com/lookfox/codeleap-test.git


Install dependencies:
npm install

Run the project:
npm run dev


The application will run at:
http://localhost:5173
